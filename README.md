## Getting Started

Clone the repo:

```bash
git clone https://github.com/nurbxfit/tvello.git
cd tvello
```

Install dependencies:

```bash
composer install
npm install
```
Run migrations:
```bash
php artisan migrate
```

Start the dev servers:
```bash
php artisan serve
```
or 
```bash
composer run dev
```

Now open http://localhost:8000
